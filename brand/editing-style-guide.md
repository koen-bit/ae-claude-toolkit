# Editing Style Guide

**Dan Martell YouTube Branding Guide**

Last Updated: July 31st, 2025 @Nic 

---

## 🎯 Purpose

To create consistency across all videos while allowing room for editors to innovate within clear creative constraints. Viewers should feel like every video came from the same editor.

---

## Core Brand Identity

- **Tone:** Premium, Intentional, Simple, minimalist
- **Editing Style Inspirations:** [Iman Gadzhi](https://www.youtube.com/@ImanGadzhi), [Johnny Harris](https://www.youtube.com/@johnnyharris), [Dylan Reynolds](https://youtu.be/O0qkkexHqZw?si=yuc8QCGChb8qX8Hq),

# Typography

### Text Style

- **Primary Font:** BBNonamePro-Bold (sentence case)
- **Secondary fonts:** Bodoni 72-Book italic (sentence case), Akkordeon-Nine (all caps)
- **Use Case:** Titles, lower thirds, and key emphasis text (aim for 80:20 primary:secondary)
- **Subtitles:** Primary font & add adjustment layer

# Music & Sound Design

### Music

- **Music Genres:** Synthwave, cinematic, ambient, light trap only when intentional
    - Example:
- **Avoid:** Generic corporate music, overly dramatic trailer music, fast-paced montage tracks
    - Example:

### Sound Design

- **Principle:** Subtle but intentional. Should feel weird if removed.
- **Style:** Cinematic, realism
- **Include:** Ambience, risers, low hits, soft whooshes
- **Avoid:** pops, swipes, overly digital effects

# Creative Freedom vs Consistency

- Visual deviation should be intentional, done with a purpose and justified by content or format.
- Editors should not spend days experimenting with radically new styles unless approved.
